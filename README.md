Ninja Arena
===========

The Open Source Multiplayer Arena game!
Currently in early development.



Current Goals 
-------------

- [x] 2D grid
- [ ] Your player is displayed at center of the screen.
- [ ] Scrolling grid
- [ ] Chat appears above players' heads
- [ ] Click/press grid location to move there.
